# Bicho_Full-Jogo-Educacional
Sobre a aplicação:

• Este projeto é um desenvolvimento de uma aplicação com fins educacionais, desenvolvida no Institito Federal do Amazonas (IFAM).

# Ferramentas de desenvolvimento
• Front end: Angular

• Back end: Spring Boot (Java)

• Banco de dados: Mysql.
